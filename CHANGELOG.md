# Changelog

## 0.2.1 - 2026-05-05

- Возвращены нативные браузерные датапикеры.
- Добавлен GitHub Actions workflow для тестов, сборки и публикации Docker-образа в GHCR.
- В README добавлен публичный адрес проекта `https://salary.tupoybot.ru`.

## 0.2.0 - 2026-05-05

- В колонке НДФЛ теперь отображаются применённые ставки и эффективная ставка месяца.
- Добавлены больничные: ввод периодов, уменьшение окладной части за рабочие дни болезни и расчёт пособия по стажу.
- Добавлены поля стажа и дохода за два предыдущих года для расчёта больничных.
- Удалён `docker-compose.yml`; запуск описан через прямые команды `docker build` и `docker run`.

## 0.1.1 - 2026-05-05

- Убрано искусственное добавление зарплатного остатка декабря прошлого года в январь выбранного года.
- Зарплата за декабрь текущего года теперь попадает в таблицу только при фактической дате выплаты в текущем году.
- Обновлены README и тесты под новую модель выплат.

## 0.1.0 - 2026-05-05

- Добавлен Flask-сервис с Docker-запуском.
- Реализован расчёт аванса, зарплатной части, отпускных, премий и прогрессивного НДФЛ.
- Подключён производственный календарь isDayOff с запасным weekday-календарём на случай сетевой ошибки.
- Добавлен интерфейс с автопересчётом, локальным сохранением данных в браузере и годовой таблицей выплат.
- Добавлены README, changelog и базовые тесты расчётного модуля.
